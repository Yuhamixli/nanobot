# GFIX通用金融信息传输系统

## 一、系统概述

**GFIX（通用金融信息传输系统）**是由**中国人民银行清算总中心**建设，**城银清算服务有限责任公司**牵头运营的境内金融机构间报文传输系统。

被誉为**"中国版SWIFT"**，是人民银行贯彻落实中央金融工作会议精神、推进金融"五篇大文章"的具体举措。

---

## 二、核心功能

| 功能 | 说明 |
|------|------|
| **统一网络接入** | 金融机构一点接入，互联互通 |
| **统一身份认证** | 标准化安全认证机制 |
| **统一报文标准** | 规范化的报文格式 |
| **7×24小时服务** | 全天候不间断运行 |
| **报文直接转发** | 支持金融机构间报文实时传输 |

---

## 三、司库信息传输业务场景

### 3.1 适用对象
- 企业集团财务公司
- 商业银行
- 其他金融机构

### 3.2 主要业务功能
- **账户查询**：实时查询银行账户余额及交易明细
- **资金划拨**：对外支付、资金归集等
- **信息交互**：与多家金融机构实现一点接入、多方互联

### 3.3 业务优势
1. **安全可靠**：央行级金融基础设施
2. **高效便捷**：实时报文传输，降低沟通成本
3. **经济适用**：相比传统银企直连成本更低
4. **自主可控**：国产替代方案，保障金融安全

## 五、与航空工业集团财务公司的关系

GFIX系统为集团司库建设提供底层报文传输基础设施：
- 夯实司库系统底层架构
- 提升企业资金管理的质量与效率
- 支持司库资金监控、预警及决策系统建设
